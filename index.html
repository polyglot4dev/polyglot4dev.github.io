Polyglot4dev conference

The first step to become polyglot developer

Nayden Gochev April 15, 2022

This is a unique conference, first of its kind conference, where visitors already familiar with a particular language or technology can actually LEARN in a fast way new language and technology!

The goal is to improve ourselves, to not be focused only on one technology stack, to be able to use the best tools for the job, and to be able to think differently and see the bigger picture.

Technology stacks come and goes, the same happens for frameworks, languages, and tools, but we are the ones that stay and wanna choose. Always wanna learn, but you never wanna lose !

So if you always wanna go, but you never wanna stay. Always want to learn new things, want to get out of your conform zone. Join us on a UNIQUE journey where we will explore new worlds, new languages, frameworks, package managers and etc.

Become a polyglot developer TODAY!

The First edition is for Java developers that want to learn other programming languages and technology stacks.

2022 - the year of the first polyglot4dev: Java Edition conference or shortly said polyglot4j ;)

It will happen in person only, the 6th of October, 2022! If you are a person who wants to learn how to write React or want to learn WTF is Dart and Flutter or why is this buzz about Kotlin (which some thing will replace Java) or wants to understand what this black magic called webpack is or wtf is this crap npm. , then this is the conference for YOU !
