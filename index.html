About the polyglot4dev conference

Nayden Gochev April 15, 2022

This is a unique conference, where visitors already familiar with particular language or technology can actually LEARN in a fast way new language and technology !

The goal is to improve ourselves, to not be focused only on one technology stack, to be able to use the best tools for the job, to be able to think differently.

Technology stacks comes and goes, the same happens for frameworks, languages, tools, but we are the ones that stay and wanna choose. Always wanna play, but you never wanna lose !

So if you always wanna go, but you never wanna stay. Always want to learn new things, want to get out of your conform zone. Join us on a UNIQUE journey where we will explore new worlds, new languages, frameworks, package managers and etc.

First edition is toward Java developers that wants to learn other programming languages and technologies. If you are a person who wants to learn how to write React or WTF is Dart and Flutter or things like Kotlin (which might replace Java) or wants to understand what this black magic called webpack is. This is the conference for YOU !

2022 - the polyglot4dev: Java Edition or shortly said polyglot4j ;)
